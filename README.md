# 客户管理系统

## 概述

![PageView](https://github.com/Avicii4/crm6/blob/master/src/main/resources/view.jpg)

一个使用 Spring 4.x + SpringMVC + Mybatis 3.x + Maven 实现的信息管理系统。

IDE：IntelliJ IDEA 2018.3.6 (Ultimate Edition)

## 实现功能

* 输入客户名称模糊查询；
* 根据顾客来源、行业、和级别分类查询；
* 数据回显；
* 查询结果的分页；
* 使用 Lombok 简化 POJO 的开发；
* JSON 数据的请求和响应。
